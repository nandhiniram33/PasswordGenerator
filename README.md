# 🔐 Password Generator (Java)

**📌 Overview**

The Password Generator is a simple Java console application that generates strong and random passwords based on the length specified by the user. It combines uppercase letters, lowercase letters, numbers, and special characters to create secure passwords, helping users improve their password security.

This project demonstrates the use of core Java concepts such as loops, strings, random number generation, and user input handling to build a practical utility.

**✨ Features**

•Generates secure random passwords

•User can specify password length

•Uses a mix of:

•Uppercase letters (A–Z)

•Lowercase letters (a–z)

•Numbers (0–9)

•Special characters (!@#$%^&*)

•Simple console-based interface

•Lightweight and easy to run

**🛠️ Technologies Used**

1.Java

2.Random class for generating random characters

3.Scanner class for user input

**⚙️ How It Works**

•The program asks the user to enter the desired password length.

•It combines different character sets including letters, numbers, and special characters.

•Random characters are selected until the password reaches the specified length.

•The generated password is displayed in the console.
